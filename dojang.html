Trust Layer

도장.
오프체인을 온체인으로.

GIWA Dojang은 신뢰할 수 있는 오프체인 정보를 검증된 온체인 어테스테이션(Attestation)으로 발급하는 GIWA Chain의 네이티브 신뢰 레이어입니다. SOVE는 모든 실물 자산의 흐름을 도장으로 봉인합니다.

01 — Concept

도장은 무엇인가.

도장(Dojang)은 승인된 발급자(Issuer)가 발급하는 온체인 어테스테이션입니다. 한국 전통의 인장이 신원과 결정을 봉인하듯, 디지털 도장은 오프체인 데이터의 진위를 온체인에 영구 기록합니다.

예를 들어 Identity Stamp는 KYC를 거친 지갑임을 증명하는 SBT 형태로 발급됩니다. 현재 GIWA 생태계에서 Identity Stamp는 Upbit Korea가 유일한 공식 발급자입니다. 같은 원리로 SOVE는 감정기관·수탁기관·보험사가 발급하는 추가 도장으로 실물 자산의 전 생애주기를 신뢰 체인 위에 올립니다.

03 — Stamp Types

SOVE에서 발급되는
6종 도장.

각 도장은 승인된 발급자만 생성할 수 있으며, 발급 시점·발급자 서명·대상 자산 ID가 온체인에 봉인됩니다. 모든 도장은 SBT(Soul Bound Token) 형태로 양도 불가합니다.

01
Appraisal Stamp
발급자 · 인증 감정기관
감정가(USD), 감정일, 감정기관 서명, 감정 방법론을 온체인에 영구 기록. $SOVE Mint의 직접 트리거가 됩니다.
02
Custody Stamp
발급자 · 인증 수탁기관
보관 위치, 보관 조건(온/습도), 입고 시점, 실물-NFT 1:1 매칭을 봉인. 월간 실사 후 갱신됩니다.
03
Insurance Stamp
발급자 · 보험사
보험 증서 번호, 커버리지(감정가 110%), 유효기간, 보험사 서명. 만료 전 자동 갱신 알림.
04
Identity Stamp
발급자 · Upbit Korea
GIWA Verified Address — KYC 완료 SBT. 1회 발급으로 모든 dApp에서 KYC 자동 통과.
05
Redemption Stamp
발급자 · 수탁기관
실물 반환 청구 시 발급. 출고 확인, 배송 정보, 인도 완료를 봉인. NFT 영구 소각의 근거.
06
Re-Appraisal Stamp
발급자 · 감정 위원회
연 1회 정기 재감정. 시장가 보정 + 담보율 재계산. 특별 재감정은 보유자 요청으로 진행.
04 — Data Structure

도장 어테스테이션 구조.

DojangAttestation.sol
struct Attestation {
  bytes32 schemaUID;       // 도장 종류 식별자
  address issuer;          // 인증된 발급 기관
  address subject;         // 대상 자산 또는 지갑
  uint256 assetId;         // SOVE 자산 ID
  uint256 issuedAt;        // 발급 timestamp
  uint256 expiresAt;       // 만료 timestamp (0 = 무기한)
  bytes32 dataHash;        // 오프체인 데이터 해시
  string  metadataURI;     // IPFS URI
  bytes   signature;       // 발급자 서명
}

검증 로직

01스키마 검증schemaUID가 등록된 도장 종류와 일치하는지 확인
02발급자 권한issuer가 해당 도장에 대한 발급 권한 보유 확인
03유효기간expiresAt > block.timestamp
04서명 검증signature가 issuer의 비공개키로 서명되었는지 ecrecover로 확인
05데이터 해시metadataURI 내용 SHA-256 = dataHash 일치 확인 → 유효
