04 — Module

Ticket Center.

ERC-1155 기반 NFT 티켓. GIWA ID에 바인딩되어 암표가 원천적으로 차단되며, 공연 종료 후 기념 컬렉터블로 자동 전환됩니다.

티켓 발행 프로세스

01파트너십 체결공연 기획사와 SOVE 계약
02공연 정보 등록일시, 장소, 좌석 배치, 등급, 가격
03NFT 티켓 발행ERC-1155 좌석별 메타데이터
041차 판매고정가 · GIWA ID 바인딩
052차 거래가격 상한선 설정 가능
06QR / NFC 체크인공연 당일 본인 확인
07기념 NFT 전환공연 종료 후 자동 컬렉터블 발행

암표 방지 5단 메커니즘

가격 상한선
기획사가 설정하는 재판매 최고가 (예: 원가의 150%). 상한 초과 거래는 컨트랙트에서 reject.
GIWA ID 바인딩
티켓 구매 시 SBT 연결 — 1인 N매 제한 강제. 양도 시 양도자/양수자 모두 KYC 통과 필수.
전송 제한
공연 48시간 전부터 전송/재판매 차단. 컨트랙트 레벨에서 강제되어 우회 불가.
체크인 본인 확인
QR 코드 스캔 + GIWA ID 일치 검증. 두 가지가 동시에 일치해야 입장.
자동 로열티
재판매마다 아티스트에게 2% 자동 분배. 암표 행위가 아티스트 수익으로 환원되는 구조.
